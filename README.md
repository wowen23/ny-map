# Manhattan Buildings Map

An interactive OpenStreetMap visualization of Manhattan buildings showing:
- Building addresses
- Year built
- Number of stories

## View the Map

Open `manhattan_map.html` in your browser or visit the GitHub Pages site.

## Data Source

Building data extracted from NYC PLUTO (Primary Land Use Tax Lot Output) dataset.

## Files

- `manhattan_map.html` - Interactive map viewer
- `manhattan_buildings.json` - Extracted building data (42,132 buildings)
- `extract_manhattan_data.py` - Python script to extract data from PLUTO CSV
